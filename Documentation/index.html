A) Set Up Your Environment - top

Before you use this template, be sure you have your development environment set up. You need to:

  1. Download Android Studio.
  2. Download the latest SDK tools and platforms using the SDK Manager.

    You can launch the SDK Manager in one of the following ways:

    Here's an outline of the packages required:
    • SDK Tools
    • SDK Platform-tools
    • SDK Platform
    • System Image
    • Android Support Library
    • Android Support Repository
    • Google Play services
    • Google Repository

B) Import Project to Android Studio - top

Start Android Studio and from the Welcome screen, select Import project. Select the DELI project folder and click Ok.

And wait for a moment to Gradle build is finishing.

C) App Structure - top

This template composed of eight Activities which interact with the user

  1. MainActivity: it serves as the main entry point to your app's user interface, and handles left DrawerLayout with NavigationView and the "MainFragment" with RecyclerView and Adapter for providing the view that represent all categories of the menu in a data set.
  2. MenuActivityWithTabs: this is the menu for the restaurant which shows the user all the categories using TabLayout and Fragments with ViewPager for the swipe tabs feature.
  3. HotDealsActivity: it handles two custom Fragments, "HotDealsListFragment" with RecyclerView and Adapter for providing the view that represent hot deal items in a data set. and "HotDealsDetailFragment" that displays the details of each hot deal items.
  4. LocationActivity: it handles the Google maps and locations integration.
  5. MyCartActivity: it handles two custom Fragments, "MyCartFragment" that shows the content of user cart via RecyclerView, and the user will be able to delete an item, or edit it, and "MyCartCheckoutFragment" that display the user information from user table before the user can submitting his order.
  6. ProfileActivityWithTabs: it handles two tabs using TabLayout with two custom Fragment, "FavoritesFragment" that shows the content of favorite table via RecyclerView, and "OrdersHistoryFragment" that shows all orders history from items table.
  7. RegisterActivity: all the fields required to create an account.
  8. LoginActivity: all the fields required to login.

I've included one pdf file (Deli App Structure) with this template shows you the various ways the user can move throughout the application.

D) Get a Google Maps API key - top

Your application needs an API key to access the Google Maps servers. The type of key you need is a Key for Android applications. The key is free. You can use it with any of your applications that call the Google Maps Android API, and it supports an unlimited number of users.

To get your API key:

  1. Get information about your app's certificate, known as its SHA-1 fingerprint. To display a certificate's SHA-1 fingerprint using the keytool program with the -v parameter.

    Enter the following at a terminal or command prompt:

    keytool -list -v -keystore your_keystore_name -alias your_alias_name

    Replace your_keystore_name with the fully-qualified path and name of the keystore, including the .keystore extension. Replace your_alias_name with the alias that you assigned to the certificate when you created it.

    You should see output similar to this:

  2. Select or create a project in the Google Developers Console if you don't already have one.
  3. Add the Google Maps Android API as a service for the project.
  4. Go to Credentials to get an Android key and set the API credentials.
  5. In the resulting dialog, enter your app's SHA-1 fingerprint and package name.
  6. Your new Android key appears in the list of API keys for your project.
  7. Add the key to your app by adding it in the string resource file and re-build your application.
